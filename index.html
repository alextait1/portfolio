About Me

Why I'm a Developer

I'm enamoured with the marriage of creativity and technique that goes into programming. I love building websites and apps that prioritize user experience, accessiblity, elegent design and brand identity.

How I Found Code

My web development journey began with building a few Squarespace sites for bands I worked with. After doing some online coding turorials, I enroled in Hacker You's part-time web development class, which was the best decision I've ever made! This summer I completed their immersive web development bootcamp. It was nothing short of a game changer for me in terms of leveling up my skills and inspiring me to push the limits of what I can imagine and bring to life on the web.

About This Project

This quiz project uses react composition, javascript and Ajax API calls to help the user determine who their Fairy Drag Mother is from the RuPaul's drag race winners endpoint and dynamically renders the results.

I used the recently published No Key, No Shade API created by fellow Hacker You graduates as a tool to explore the world of react, creating an app that is focused on a fun user experience and embodying RuPaul's brand.
